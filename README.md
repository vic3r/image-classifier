# Image Classifier

web implementation for classifying certain kind of images (fruits) based on their features.
## Getting Started

This is a tool which will help to classify fruit images


### Prerequisites

```
install node 8.11
```

In client and server

```
run npm install
```

### Installing

Run frontend

```
gatsby develop
```

Run backend

```
npm start
```
